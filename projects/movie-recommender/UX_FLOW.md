# UX Flow

## Flow 1: First-time user
1. arrives on WatchTonight
2. sees simple product promise
3. selects streaming services
4. selects taste preferences
5. optionally selects mood
6. gets 3 movie picks + one best overall pick

## Flow 2: Returning user
1. returns to product
2. existing taste profile is reused
3. optionally adjusts mood / tonight's preference
4. gets updated recommendations

## Flow 3: Feedback loop
1. user watches a movie
2. user gives quick feedback
3. profile updates over time
4. future recommendations improve

## UX rule
Every step should feel lighter than browsing Netflix for 20 minutes.
