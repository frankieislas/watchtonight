# 7-Day Operating Cadence

## Monday - Set the bets
- choose top 1 to 3 experiments
- assign ownership
- define success criteria

## Tuesday to Thursday - Execute hard
- ship
- recruit testers
- run outreach
- collect early signals

## Friday - Read the data
- update scoreboard
- review usage, signups, retention, money signals
- decide what is real vs noise

## Saturday - Tighten and document
- polish what is working
- document lessons
- remove weak or distracting work

## Sunday - Decide next cycle
- keep, kill, or pivot experiments
- define next week's top bets

## Rule
Every cycle should end with:
- one clearer belief
- one stronger asset
- one higher-confidence bet
