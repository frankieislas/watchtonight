# Email-Only Signup Flow

## Goal
Turn WatchTonight into a real waitlist page that collects email signups with minimal friction.

## Form behavior
- one required field: email
- one primary CTA button
- success state after submit

## Recommended UX
### Input
Placeholder:
Enter your email

### CTA
Join the early access list

### Success message
You're in.

WatchTonight is being built to help you skip the endless scrolling and get 3 strong movie picks you can actually watch tonight.

We'll let you know when early access opens.

## Basic requirements
- email input validation
- visible success state
- no additional required fields
- no unnecessary friction

## Optional next additions later
- first name
- favorite streaming services
- biggest movie-picking frustration

## Validation principle
At this stage, the only real question is whether people want this enough to give their email.
