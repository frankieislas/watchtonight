# WatchTonight Tech Stack Recommendation

## Goal
Choose the smallest practical technical stack for the first WatchTonight web MVP.

## Recommendation
Use a lightweight web stack centered on:
- Next.js
- React
- simple CSS or Tailwind-level styling
- lightweight server-side/API route logic only where needed

## Why this is the recommendation
- fast to build and iterate
- strong fit for a simple web MVP
- easy to host
- supports both landing page and product UI in one codebase
- flexible enough for recommendation inputs, result pages, and lightweight profile memory later

## What this stack should cover in V1
- landing page
- recommendation input flow
- recommendation output page
- lightweight feedback input
- simple product shell for early beta users

## What not to overbuild yet
- heavy backend architecture
- complex auth system
- large database design before needed
- microservices
- mobile app codebase

## Practical interpretation
Use the stack to ship the first useful version quickly, not to build the final architecture.
