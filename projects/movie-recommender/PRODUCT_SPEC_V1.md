# WatchTonight Product Spec V1

## Product summary
WatchTonight is a personalized movie recommendation product that helps people stop endlessly browsing streaming apps and quickly get 3 strong movie picks they can watch tonight.

## User problem
Users have plenty of streaming access but too many options. They want a fast, trustworthy answer, not another giant catalog.

## Core promise
Give the user 3 strong movie options that fit their streaming services and taste, plus one best overall recommendation.

## Primary user inputs
- streaming services available
- genres liked
- genres disliked
- optional mood / vibe
- optional watched/rated feedback over time

## Primary outputs
- 3 movie recommendations
- where each can be watched
- why each recommendation fits
- one best overall pick

## Taste memory
The product should learn from:
- watched movie feedback
- likes/dislikes
- tone preferences
- pacing preferences
- repeated signals over time

## UX principles
- fast
- clear
- low-friction
- high-signal
- no overload

## Success criteria
A user should be able to:
- understand the product quickly
- give basic preferences quickly
- get recommendations quickly
- feel more confident in what to watch

## Not in V1
- TV recommendation engine
- deep social/community features
- advanced watchlist management
- native mobile apps

## Open product questions
- how much onboarding is enough before recommendations feel useful?
- should taste setup be one screen or multiple?
- should mood selection be optional or prominent?
- should feedback happen immediately after recommendations or later after watching?
