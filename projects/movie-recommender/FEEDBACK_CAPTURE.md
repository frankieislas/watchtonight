# Early Feedback Capture

## Goal
Capture lightweight feedback from the first soft-launch recipients without creating a heavy process.

## Core question
- Is this useful enough that you would sign up or share it?

## Optional follow-ups
- What is the first thing that confused you?
- What would make this more compelling right away?
- Would you personally use this?

## Signals to track
- signed up immediately
- said the problem feels real
- asked follow-up questions
- shared confusion
- lukewarm response
- strong excitement

## Interpretation rule
Prioritize:
- actual signups
- clear understanding
- repeated patterns in feedback

Do not overweight:
- vague encouragement
- one-off edge-case preferences
- compliments without intent
