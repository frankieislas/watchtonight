# Build Order

## Recommended order
1. define MVP spec clearly
2. define UI/UX flow
3. choose technical stack for the first web app
4. build input screen
5. build recommendation output screen
6. wire basic recommendation logic
7. add lightweight taste memory
8. test with early users

## Immediate next build question
What technical stack should the first web MVP use?

## Recommendation
Choose the smallest stack that makes iteration fast.
Avoid over-architecting before real usage.
