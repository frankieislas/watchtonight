# Business OS - Rollout Sequence

Create or repurpose the core Discord channels:
- command-desk
- scoreboard
- weekly-review
- scout
- builder
- closer
- analyst
- operator
- archivist
- distribution
- user-feedback
- offers

## Step 2
Pin the operating docs in the relevant places.

## Step 3
Start using the command format for top-priority updates.

## Step 4
Move active project decisions into command-desk and scoreboard instead of letting them stay fragmented.

## Step 5
Run the first 7-day cycle with the weekly system.

## Step 6
After one cycle, prune anything that adds noise without improving speed, clarity, or money signals.
